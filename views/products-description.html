Description

A software that supports the production of patient individual „Medifilms“ covering the entire business process.
With the portal technology i-medify AG is able to provide a technical platform for different application solutions or services and accommodate them into a single integrated interface. This can e.g. be sharing of the Medifilm valuation (Medicares) and a home care solution.

Medicares

Support the whole order process of "Medifilms" ranging from its initial order, production and the corresponding delivery to the purchaser.
The following application components are available:

  • Set up and manage different organizational structures
  • Therapy survey, processing and management
  • Order data generation and processing
  • Production data generation and processing
  • Invoicing and extra accounting like billing for health insurance, narcotics
  • Shipment planning and execution
  • Procurement management and execution
  • Inventory management
  • Multi-Tenant User Management
  • System auditing

Process overview

  1. Keep production medicaments upto date
  2. Create therapy online
  3. Therapy survey and validation
  4. Release order proposal
  5. Process order in production
  6. Adjust therapy online
  7. Provide invoices and accounting forcustomer
  8. Prepare shipping documents for delivery
